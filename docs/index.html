RP-Initiated Logout (GET)

OIDC RP-Initiated Logout per OpenID Connect RP-Initiated Logout 1.0 §2. Clears the IdP session and optionally redirects to post_logout_redirect_uri.

query Parameters
id_token_hint
string

Previously issued ID token

post_logout_redirect_uri
string

URI to redirect to after logout

state
string

Opaque value passed back to post_logout_redirect_uri

client_id
string

Client identifier (used to validate post_logout_redirect_uri when no id_token_hint)

Responses

https://localhost:9999/oauth2/logout
